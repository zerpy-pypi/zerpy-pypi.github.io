About ZerPy

ZerPy is a Python library that simplifies development by providing tools for database management, environment variable handling, and utility functions. Developed by MohamedLunar, ZerPy is perfect for developers seeking efficiency and ease.

Features

  • Manage databases seamlessly (SQLite, MySQL, PostgreSQL).
  • Load and manage environment variables easily.
  • Access various utility functions for string manipulation and validation.
