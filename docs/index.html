Introduction

ZerPy is a powerful Python library that helps developers manage databases, environment variables, and utilities in a simple and efficient way.

Installation

Install ZerPy using pip:

pip install zerpy

Usage

Here's an example of how to use ZerPy for database and envirenements management:

import zerpy, os
from zerpy import db, env
db = zerpy.connect("example.db")
db.load()
env.load()
API_KEY = os.getenv("API_KEY")
